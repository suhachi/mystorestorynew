## 📋 사용자 작업 필요 사항

### 🔴 필수 작업 (배포 전)

1. **환경 변수 설정**
   - Firebase Console 접속: https://console.firebase.google.com
   - 프로젝트 선택: `mystorestory`
   - 프로젝트 설정 > 일반 > 웹 앱에서 SDK 설정 확인
   - `.env.local` 파일 생성 및 값 입력
   - 참고: `환경-변수-설정-가이드.md`

2. **Firebase Console 설정 확인**
   - Authentication 활성화 확인
   - Firestore Database 생성 확인
   - Storage 활성화 확인
   - Hosting 활성화 확인
   - Functions 활성화 확인
   - 참고: `Firebase-프로젝트-설정-확인-보고서.md`

3. **빌드 테스트 실행**
   - `.env.local` 파일 확인
   - `npm install` 또는 `pnpm install` 실행
   - `npm run build` 실행
   - 빌드 성공 확인
   - 참고: `빌드-테스트-가이드.md`

### 🟡 선택 작업 (배포 후 개선)

1. **CI/CD 파이프라인 설정**
   - GitHub Actions 워크플로우 생성
   - 자동 배포 설정

2. **모니터링 설정**
