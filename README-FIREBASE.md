- **프로젝트 ID**: mystorestory
- **프로젝트 번호**: 102904078280

## 시작하기

### 설치

```bash
pnpm install
```

### 개발 서버 실행

```bash
pnpm dev
```

### 빌드

```bash
pnpm build
```

### 배포

```bash
pnpm deploy
```

## Firebase 서비스

- Authentication
- Firestore
- Storage
- Analytics
- Hosting
