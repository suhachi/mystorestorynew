### 2. 인코딩 스크립트 (scripts 폴더)

#### 체크 및 검증 스크립트
- ✅ `scripts/encoding-check.ps1` - 인코딩 설정 체크 스크립트
  - Git 전역 설정 확인
  - 프로젝트 설정 파일 확인
  - 파일 줄바꿈 문자 샘플 체크
  - 한글 인코딩 테스트

- ✅ `scripts/check-korean-content.ps1` - 한글 콘텐츠 인코딩 검증
  - 중요 파일의 한글 콘텐츠 검증
  - 줄바꿈 문자 검사
  - 손상 감지

#### 수정 및 복구 스크립트
- ✅ `scripts/encoding-fix.ps1` - 인코딩 수정 스크립트
  - 손상된 파일의 인코딩 수정
  - UTF-8로 변환

- ✅ `scripts/fix-encoding.ps1` - 인코딩 복구 스크립트
  - 특정 파일의 인코딩 복구
  - 손상 패턴 복구

#### 설정 자동화 스크립트
- ✅ `scripts/setup-encoding.ps1` - 인코딩 설정 자동화
  - Git 전역 설정 자동화
  - 필수 설정 파일 확인
  - 기존 파일 정규화
  - VS Code 확장 확인

**위치**: `MY_STORE_STORYdesign/scripts/`

**사용법**:
```powershell
# 인코딩 설정 체크
.\scripts\encoding-check.ps1

# 한글 콘텐츠 검증
.\scripts\check-korean-content.ps1

# 인코딩 설정 자동화
.\scripts\setup-encoding.ps1

# 인코딩 수정
.\scripts\encoding-fix.ps1

# 특정 파일 복구
.\scripts\fix-encoding.ps1
```

## 📝 스크립트 설명

### encoding-check.ps1
**목적**: 인코딩 설정 전반적인 체크

**기능**:
- Git 전역 설정 확인
- 프로젝트 설정 파일 확인
- 파일 줄바꿈 문자 샘플 체크
- 한글 인코딩 테스트

**사용법**:
```powershell
.\scripts\encoding-check.ps1
```

### check-korean-content.ps1
**목적**: 한글 콘텐츠 인코딩 검증

**기능**:
- 중요 파일의 한글 콘텐츠 검증
- 줄바꿈 문자 검사
- 손상 감지

**사용법**:
```powershell
.\scripts\check-korean-content.ps1
```

### encoding-fix.ps1
**목적**: 인코딩 수정

**기능**:
- 손상된 파일의 인코딩 수정
- UTF-8로 변환

**사용법**:
```powershell
.\scripts\encoding-fix.ps1
```

### fix-encoding.ps1
**목적**: 특정 파일 인코딩 복구

**기능**:
- 특정 파일의 인코딩 복구
- 손상 패턴 복구

**사용법**:
```powershell
# 스크립트 내부에서 파일 경로 수정 후 실행
.\scripts\fix-encoding.ps1
```

### setup-encoding.ps1
**목적**: 인코딩 설정 자동화

**기능**:
- Git 전역 설정 자동화
- 필수 설정 파일 확인
- 기존 파일 정규화
- VS Code 확장 확인

**사용법**:
```powershell
# 전체 설정
.\scripts\setup-encoding.ps1

# Git 설정 건너뛰기
.\scripts\setup-encoding.ps1 -SkipGitConfig

# 정규화 건너뛰기
.\scripts\setup-encoding.ps1 -SkipNormalization
```
