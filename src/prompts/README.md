## 🌟 소개

MyStoreStory는 **배달 수수료 0원**으로 자체 배달앱을 만들 수 있는 노코드 플랫폼입니다.

이 저장소는 MyStoreStory의 **완전한 구현 가이드**를 담고 있는 **109개의 ATOMIC 프롬프트 시스템**입니다.

### ✨ 특징

- 🎯 **109개의 세밀한 프롬프트**: 프로젝트 초기화부터 프로덕션 런치까지
- 📚 **완전한 구현 가이드**: 40개 파일, 65+ 컴포넌트, 20+ 페이지
- 🚀 **즉시 사용 가능**: 복사-붙여넣기로 바로 구현
- 💯 **프로덕션 준비**: 테스트, 배포, 모니터링 포함
- 📖 **완벽한 문서화**: 사용자 가이드부터 API 레퍼런스까지

## 💻 기술 스택

### Frontend
- **React** 18.3.1 - UI 라이브러리
- **TypeScript** 5.x - 타입 안정성
- **Vite** 6.0.1 - 빌드 도구
- **Tailwind CSS v4** - 스타일링
- **Shadcn/ui** - UI 컴포넌트

### Backend
- **Firebase Authentication** - 사용자 인증
- **Cloud Firestore** - 데이터베이스
- **Cloud Functions** - 서버리스 함수
- **Cloud Storage** - 파일 저장
- **Firebase Hosting** - 호스팅 & CDN

### 외부 API
- **KG Inicis** - 결제
- **Kakao Maps** - 지도
- **Google/Kakao Login** - 소셜 로그인
- **FCM** - 푸시 알림

### 도구
- **ESLint + Prettier** - 코드 품질
- **Vitest** - 유닛 테스트
- **Playwright** - E2E 테스트
- **Lighthouse** - 성능 측정

- ✅ 실시간 미리보기

#### 5. Config Modals (7개)
- ✅ Dashboard 설정
- ✅ Menu 설정
- ✅ Order 설정
- ✅ Customer 설정
- ✅ Analytics 설정
- ✅ Points 설정
- ✅ Settings 설정

#### 6. Store Admin Dashboard (7 pages)
- ✅ 대시보드 홈 (KPI, 차트, 최근 주문)
- ✅ 메뉴 관리 (CRUD, 카테고리, 옵션)
- ✅ 주문 관리 (실시간, 상태 변경, 히스토리)
- ✅ 고객 관리 (목록, 세그먼트, 포인트)
- ✅ 분석 (매출, 인기 메뉴, 트렌드)
- ✅ 설정 (상점 정보, 영업 시간, 결제)

#### 7. Customer App (4 pages)
- ✅ 메뉴 탐색 (카테고리, 검색, 필터)
- ✅ 장바구니 & 결제
- ✅ 주문 추적 (실시간 상태)

#### 8. Admin Dashboard (4 pages)
- ✅ 사용자 관리
- ✅ 상점 관리
- ✅ 분석 대시보드
- ✅ 시스템 설정

- ✅ useAnalyticsConfig - 분석 설정
- ✅ usePointsConfig - 포인트 설정
- ✅ useSettingsConfig - 설정
- ✅ useFeatureCards - Feature Cards
- ✅ useDragAndDrop - 드래그 앤 드롭

#### 10. Cloud Functions (6개)
