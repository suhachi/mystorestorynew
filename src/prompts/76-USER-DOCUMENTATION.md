### 1. 시작하기

#### 회원가입

**웹사이트 접속**
```
https://mystorestory.com
```

**회원가입 단계**:

1. **"시작하기" 버튼 클릭**
   - 우측 상단 또는 메인 페이지 중앙

2. **정보 입력**
   ```
   - 이메일: example@email.com
   - 비밀번호: 8자 이상, 영문+숫자 조합
   - 이름: 홍길동
   - 연락처: 010-1234-5678
   ```

3. **이메일 인증**
   - 받은 메일함 확인
   - 인증 링크 클릭
   - 인증 완료

4. **프로필 설정**
   - 상점 이름
   - 업종 선택
   - 주소 입력

#### 첫 로그인

```
1. 이메일, 비밀번호 입력
2. "로그인" 클릭
3. 대시보드로 이동
```

### 3. 메뉴 관리

#### 메뉴 추가하기

**단계**:

1. **대시보드 → 메뉴 관리**
2. **"새 메뉴 추가" 버튼**
3. **정보 입력**:

```yaml
메뉴명: 아메리카노
카테고리: 커피
가격: 4,500원
설명: 신선한 원두로 내린 정통 에스프레소
이미지: (업로드)
옵션:
  - ICE/HOT
  - 사이즈: Regular, Large
```

4. **"저장" 버튼**

#### 메뉴 수정

1. **메뉴 목록에서 메뉴 클릭**
2. **정보 수정**
3. **"저장"**

#### 메뉴 비활성화

```
품절 또는 판매 중지 시:
메뉴 카드 → 토글 스위치 OFF
```

### 4. 주문 관리

#### 신규 주문 확인

**알림**:
```
🔔 새 주문이 들어왔습니다!
주문번호: #12345
고객: 김철수
메뉴: 아메리카노 x 2
총액: ₩9,000
```

**처리 단계**:

1. **주문 확인**
   - 주문 내역 확인
   - 고객 정보 확인
   - 배달 주소 확인

2. **주문 승인**
   - "승인" 버튼 클릭
   - 예상 준비 시간 입력: 15분
   - 고객에게 자동 알림

3. **조리 중**
   - "조리 시작" 버튼
   - 상태: PREPARING

4. **준비 완료**
   - "준비 완료" 버튼
   - 상태: READY
   - 배달원 호출 또는 고객 픽업 대기

5. **배달 중** (배달 주문)
   - "배달 시작" 버튼
   - 상태: DELIVERING

6. **완료**
   - "완료" 버튼
   - 상태: COMPLETED

#### 주문 취소

```
취소 사유:
- 재고 부족
- 배달 불가
- 고객 요청

"취소" 버튼 → 사유 선택 → 확인
```

### 8. 자주 묻는 질문 (FAQ)

#### Q1. 앱 만드는데 얼마나 걸리나요?
**A**: 약 3-5분이면 앱이 생성됩니다. 정보 입력에 따라 10-15분 정도 소요될 수 있습니다.

#### Q2. 플랜은 언제든 변경 가능한가요?
**A**: 네, 언제든 업그레이드/다운그레이드 가능합니다. 변경 시 즉시 적용됩니다.

#### Q3. 메뉴는 몇 개까지 등록할 수 있나요?
**A**: 
- Basic: 50개
- Pro: 200개
- Enterprise: 무제한

#### Q4. 고객이 앱을 어떻게 사용하나요?
**A**: 고객은 웹 브라우저 또는 PWA로 접속하여 주문할 수 있습니다. 앱 다운로드는 필요 없습니다.

#### Q5. 결제는 어떻게 되나요?
**A**: 신용카드, 카카오페이, 토스페이 등 다양한 결제 수단을 지원합니다. PG사와 연동되어 안전하게 결제됩니다.

#### Q6. 수수료가 있나요?
**A**: 플랜 구독료 외 별도 수수료는 없습니다. 배달앱 수수료 없이 자체 운영 가능합니다.

#### Q7. 배달은 어떻게 하나요?
**A**: 자체 배달원 운영, 배달대행 서비스 연동, 고객 직접 픽업 등 선택 가능합니다.

#### Q8. 고객 데이터는 안전한가요?
**A**: Firebase 보안 시스템으로 모든 데이터가 암호화되어 안전하게 보관됩니다.
