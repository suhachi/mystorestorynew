## 🔍 왜 간단해 보이는가?

스크린샷에서 페이지가 간단해 보이는 이유:

### 1. **스크롤 전 화면**
사용자께서 제공하신 스크린샷은 **페이지 상단 일부만** 보이는 상태입니다.
실제로는 **아래로 스크롤하면 5개의 완전한 섹션**이 있습니다.

### 2. **Hero Section만 보임**
현재 화면에 보이는 것:
```
┌────────────────────────────────────┐
│ 도움이 필요하신가요?                │  ← 이것만 보임
│ MyStoreStory 사용에 관한...        │
│                                    │
│ (스크롤하면 아래 내용들 나타남)     │
│ ↓                                  │
│ [실시간 채팅] [전화] [이메일]      │  ← 안 보임
│ FAQ 섹션                           │  ← 안 보임
│ 문의 폼                            │  ← 안 보임
│ 리소스                             │  ← 안 보임
│ 운영시간                           │  ← 안 보임
└────────────────────────────────────┘
```
