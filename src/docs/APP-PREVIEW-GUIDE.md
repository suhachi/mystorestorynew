# 플랜별 앱 미리보기 가이드

**실제 고객이 보게 될 배달앱을 플랜별로 미리 확인하세요!**

---

## 🎯 빠른 시작

### 방법 1: Quick Access (추천)
```
http://localhost:5173/?route=app-preview-quick
```

**특징:**
- 4가지 플랜 카드 형태로 한눈에 비교
- 각 플랜 클릭 시 즉시 미리보기
- 주요 기능 요약 확인

### 방법 2: 전체 미리보기
```
http://localhost:5173/?route=app-preview-by-plan
```

**특징:**
- 좌측: 플랜 선택 + 기능 상세 비교
- 우측: 실시간 앱 미리보기
- 디바이스 전환 (모바일/태블릿/데스크톱)

## 📱 기능 설명

### 1. 플랜별 기능 차이

#### 🔓 FREE (무료 플랜)
```
✅ 메뉴 관리 (최대 20개)
✅ 주문 접수 (월 50건)
✅ 주문 알림
✅ 기본 통계
✅ 커스텀 로고

❌ 포인트 시스템
❌ 쿠폰 관리
❌ 고객 세분화
❌ 고급 분석
❌ 커스텀 컬러

💡 하단에 "Powered by MyStoreStory" 워터마크 표시
```

#### ⚡ BASIC (베이직 플랜)
```
✅ 메뉴 관리 (최대 100개)
✅ 주문 접수 (월 500건)
✅ 주문 알림
✅ 기본 통계
✅ 커스텀 로고
✅ 커스텀 컬러
✅ 포인트 시스템 (기본)
✅ 쿠폰 관리 (3개)

❌ 고객 세분화
❌ 고급 분석
❌ AI 추천

💡 포인트 배너, 쿠폰 섹션 활성화
```

#### ⭐ PREMIUM (프리미엄 플랜)
```
✅ 메뉴 관리 (무제한)
✅ 주문 접수 (무제한)
✅ 모든 기본 기능
✅ 포인트 시스템 (고급)
✅ 쿠폰 관리 (10개)
✅ 고객 세분화
✅ 고급 분석
✅ 로열티 티어 시스템
✅ 고급 검색

❌ AI 추천 시스템
❌ 다중 매장 관리

💡 골드/플래티넘 등 회원 등급 표시
```

#### 👑 ENTERPRISE (엔터프라이즈)
```
✅ 모든 PREMIUM 기능
✅ AI 맞춤 추천
✅ 다중 매장 관리
✅ 완전 화이트라벨
✅ API 연동
✅ 전담 지원

💡 AI 추천 섹션, 고급 개인화 기능
```

## 🎨 UI 요소 비교

### Header (상단바)
```
공통:
- 상점명
- 위치 (MapPin 아이콘)
- 알림 버튼
- 장바구니 (배지 포함)

검색바:
- FREE/BASIC: "메뉴 검색..."
- PREMIUM/ENTERPRISE: "메뉴, 카테고리, 재료 검색..."
```

### 포인트/리워드 배너
```
FREE: 표시 안 됨

BASIC:
- 내 포인트: 2,500P
- [사용하기] 버튼

PREMIUM/ENTERPRISE:
- 내 포인트: 2,500P
- 회원 등급: 골드 회원 (⭐)
- [사용하기] 버튼
```

### AI 추천 섹션
```
FREE/BASIC/PREMIUM: 표시 안 됨

ENTERPRISE:
- "AI 맞춤 추천" 헤더
- ENTERPRISE 배지
- 2개 추천 메뉴 카드
- "회원님이 좋아할 메뉴" 설명
- "98% 일치율" 등 스코어 표시
```

### 쿠폰 섹션
```
FREE: 표시 안 됨

BASIC:
- 최대 3개 쿠폰 카드
- 횡스크롤

PREMIUM/ENTERPRISE:
- 최대 10개+ 쿠폰 카드
- 횡스크롤
- 애니메이션 효과
```

### 카테고리 그리드
```
FREE: 4개 카테고리
BASIC/PREMIUM/ENTERPRISE: 8개 카테고리
```

### 하단 네비게이션
```
FREE:
- 홈
- 검색
- 주문
- MY

BASIC/PREMIUM/ENTERPRISE:
- 홈
- 검색
- 주문
- 혜택 (포인트/쿠폰)
- MY
```

## 💡 활용 시나리오

### 1. 고객 프레젠테이션
```
1. Quick Access 페이지 열기
2. "이런 플랜들이 있습니다" 소개
3. 각 플랜 카드 클릭하며 실시간 데모
4. 고객 요구사항에 맞는 플랜 추천
```

### 2. 내부 기획 회의
```
1. 전체 미리보기 페이지 열기
2. 좌측에서 플랜별 기능 상세 확인
3. 우측에서 실제 UI/UX 검증
4. 디바이스별 반응형 테스트
```

### 3. 개발자 참고
```
1. 전체 미리보기 페이지 열기
2. 각 플랜의 feature flags 확인
   - features.points
   - features.coupons
   - features.aiRecommendations
3. 조건부 렌더링 로직 확인
```

## 🔧 커스터마이징

### 플랜 정보 수정
```typescript
// /pages/app-preview-by-plan.tsx

const PLAN_FEATURES: PlanFeatures[] = [
  {
    plan: 'FREE',
    name: '무료 플랜',
    features: [
      {
        category: '기본 기능',
        items: [
          { name: '메뉴 관리', enabled: true, limit: '최대 20개' },
          // 여기에 기능 추가/수정
        ]
      }
    ]
  }
];
```

### 앱 미리보기 UI 수정
```typescript
// /components/examples/customer-app-preview-by-plan.tsx

// Feature flags 수정
const features = {
  points: plan !== 'FREE',
  coupons: plan !== 'FREE',
  // 새 기능 추가
  newFeature: plan === 'PREMIUM' || plan === 'ENTERPRISE',
};

// UI 조건부 렌더링
{features.newFeature && (
  <div>새 기능 UI</div>
)}
```
